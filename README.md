Ecos da Floresta é um jogo educativo desenvolvido utilizando HTML, CSS e JavaScript, com aplicação prática dos conceitos de Lógica de Programação e Programação Orientada a Objetos (POO). O projeto foi construído ao longo de menos de três meses de aula, sendo aproximadamente um mês dedicado exclusivamente à introdução aos fundamentos da programação. Mesmo com esse tempo reduzido de formação formal, o desenvolvimento buscou aplicar organização estrutural, clareza lógica e boas práticas desde o início.

A proposta surgiu a partir da apresentação do jogo em sala, quando foi solicitado que os alunos interpretassem o conceito original e desenvolvessem uma versão própria. O trabalho foi realizado em dupla, o que exigiu alinhamento constante de ideias, divisão de responsabilidades e revisão conjunta do código. Antes de iniciar a implementação prática, foi elaborado um planejamento detalhado contendo definição de público-alvo, objetivos educacionais, mecânicas principais, proposta narrativa e estrutura geral da experiência do jogador. Essa etapa de organização foi essencial para evitar improvisações e garantir que o desenvolvimento seguisse uma linha coerente.

Durante a implementação, foram aplicados diversos conceitos fundamentais da programação. Variáveis globais e locais foram utilizadas para controlar estado, pontuação e progresso. Estruturas condicionais foram responsáveis por definir decisões dentro do jogo, como validação de ações e controle de fluxo. Laços de repetição permitiram automatizar processos e atualizar elementos dinamicamente. Arrays foram utilizados para organizar conjuntos de dados, como desafios, perguntas ou elementos interativos. Funções foram criadas para dividir o código em blocos lógicos bem definidos, facilitando manutenção e compreensão.

O projeto também seguiu o modelo Entrada, Processamento e Saída. As interações do usuário representam a entrada; a lógica interna do sistema realiza o processamento; e a atualização visual na interface representa a saída. Essa estrutura ajudou a compreender como sistemas computacionais organizam informações e respondem a estímulos.

Além da lógica interna, houve manipulação direta do DOM para atualização dinâmica da interface, controle de eventos para capturar interações do usuário e implementação de mecanismos que garantem a fluidez da experiência. O desenvolvimento passou por múltiplas etapas de teste, correção de erros e refinamento de detalhes, fortalecendo a compreensão prática da importância da revisão constante no processo de programação.

O versionamento do projeto foi realizado utilizando Git, reforçando a prática de organização e controle de alterações no código. O projeto representa não apenas a criação de um jogo educativo, mas também a consolidação dos primeiros aprendizados em desenvolvimento web, integração entre HTML, CSS e JavaScript, organização estrutural de código e aplicação prática de conceitos teóricos estudados em sala.

Mesmo com um período curto de aprendizado formal, o desenvolvimento do Ecos da Floresta demonstra evolução técnica, capacidade de planejamento, organização lógica e compreensão dos fundamentos necessários para a construção de aplicações interativas na web.
